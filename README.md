# Course Selling Application

A full-stack MERN starter for a course marketplace with:

- Student authentication and enrollment
- Admin dashboard with role-based access
- Course creation, module management, and content uploads
- Protected course content delivery

## Tech Stack

- Frontend: React, Vite, Tailwind CSS, Redux Toolkit, React Router
- Backend: Node.js, Express.js, Mongoose, JWT, Multer
- Database: MongoDB

## Quick Start

1. Install dependencies:

```bash
npm install
```

2. Create environment files:

- Copy `server/.env.example` to `server/.env`
- Copy `client/.env.example` to `client/.env`

3. Start the app:

```bash
npm run dev
```

4. Seed an admin user:

```bash
npm run seed:admin
```
